# BGMChart-FrontEnd
BGMChart 프론트엔드(각 음원사이트 순위 종합 및 뮤직 플레이어)

# ⌛개발 기간
2023.10 ~ 2023.11

### 👯팀원
* 팀장 : 김상훈 - 메인페이지 및 플레이리스트 페이지, 채팅방 페이지 구현
* 팀원 : 사공도현 - 각 음원 비디오 크롤링 및 검색 기능 구현
* 팀원 : 김병욱 - 모든페이지 디자인 및 메인 페이지 구현

### 💾개발환경
* **WPF .NET Framework**  
* **C#**  
* **Visual Studio2022**
